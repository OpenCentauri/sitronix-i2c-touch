# refs

Reference material for the CC2 Sitronix driver work.

## Layout

- `vendor-driver/` — original Sitronix vendor `.c/.h` reference driver
- `st77922-touch-driver/` — external ST77922 touch reference from Beast12
- `cc2-stock-kernel-decomp/` — pseudo-C export of the stock CC2 kernel Sitronix driver
- `cc2-stock-analysis/` — stock CC2 captures, disassembly, kallsyms, DT extracts, and helper scripts
