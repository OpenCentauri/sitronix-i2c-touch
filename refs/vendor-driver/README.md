# Sitronix vendor driver reference

Original Sitronix vendor reference driver kept for comparison:

- `sitronix_i2c_touch.c`
- `sitronix_i2c_touch.h`

Why keep it:
- old Sitronix register/protocol reference
- feature-parity comparisons
- source provenance for the out-of-tree Linux 6.18 port

Do not build this copy directly against modern kernels.
